![](https://41k0st0.my.site.com/ayuda/servlet/rtaImage?eid=ka0Qm0000001k2P&feoid=00N8c00000H564C&refid=0EMQm00000HYvDN)

1. Ingresa desde Colombia a [https://winplay.co/plans](https://winplay.co/plans).​
2. Selecciona el Plan adquirido. ​
3. Debe Iniciar Sesión si ya está registrado ó Registrate con Email con todos tus datos personales. ​
4. Baja hasta la sección Tengo un cupón y escribe en ese espacio el código asignado. ​
5. Por último, selecciona la opción Registrar y aplicar cupón.​
6. Preferiblemente crear una cuenta nueva para usuarios antiguos o nuevos. ​
