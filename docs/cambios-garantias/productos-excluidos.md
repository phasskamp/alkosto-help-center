## ✅ ¿Qué Opciones Tengo?

Aunque estos productos no aplican para cambio voluntario, **SÍ tienen garantía** por defectos de fábrica:

| Si el producto... | Puedes... |
|-------------------|-----------|
| Llegó dañado | Reportar en 24-48h para cambio o reembolso |
| Tiene defecto de fábrica | Tramitar garantía (reparación/cambio) |
| No funciona correctamente | Tramitar garantía |
| Llegó incompleto | Reportar inmediatamente para completar |


## 🔍 Casos Especiales

### Productos Apple
Aunque Alkosto no los cambia por retracto, Apple tiene su propia política. Puedes contactar a Apple directamente si compraste hace menos de 14 días.

### Consolas y videojuegos
Si el empaque está **sellado y sin abrir**, algunos casos pueden evaluarse. Contacta a servicio al cliente.

### Productos sellados de higiene/cosméticos
Si el producto está **100% sellado** sin ninguna apertura, podría evaluarse caso por caso.


## ❓ Preguntas Frecuentes

### ¿Por qué estos productos están excluidos?
Las exclusiones existen por razones de:
*   **Higiene:** Productos de uso personal
*   **Seguridad:** Alimentos, baterías
*   **Regulación:** Licores, vehículos
*   **Tecnología:** Códigos de activación únicos
*   **Falsificación:** Consumibles como tintas

### ¿Tienen garantía estos productos?
**Sí.** Todos los productos nuevos tienen garantía legal contra defectos de fábrica, independientemente de si aplican para cambio voluntario.

### ¿Qué hago si me arrepentí de comprar un producto excluido?
Lamentablemente, el derecho de retracto no aplica para estos productos. Podrías intentar:
*   Venderlo de forma particular
*   Regalarlo
*   Contactar al fabricante (algunas marcas tienen políticas propias)

### ¿Si compré en línea cambia algo?
No. Las mismas exclusiones aplican para compras en tienda y en línea.
