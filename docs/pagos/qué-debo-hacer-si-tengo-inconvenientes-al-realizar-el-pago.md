:::tip En este artículo aprenderás:
* Identificar problemas comunes con los pagos.
* Verificar la información de tu método de pago.
* Cuándo y cómo contactar a nuestro equipo de soporte.
:::


### 1. Verifica tu Información Antes de contactar a soporte, te recomendamos revisar que los datos de tu método de pago estén correctos. Los errores más comunes son:
1. ▸ Número de tarjeta incorrecto.
2. ▸ Fecha de vencimiento inválida.
3. ▸ Código de seguridad (CVV) incorrecto.
4. ▸ Fondos insuficientes en la cuenta.

### 2. Soluciones Rápidas a Problemas Comunes Si tu información es correcta, intenta estas soluciones: | | | | --- | --- | | **Tarjeta Rechazada** | Contacta a tu banco para verificar si existen restricciones o bloqueos de seguridad. | | **La Página No Carga** | Actualiza tu navegador o intenta realizar la compra desde otro dispositivo. | | **Error de Conexión** | Asegúrate de tener una conexión a internet estable. |

### 3. Contacta con Soporte Si el problema persiste, nuestro equipo de atención al cliente está listo para ayudarte. **💡 Consejo:** Ten a mano los detalles de tu intento de compra (producto, método de pago, mensaje de error) para agilizar la atención. [Contactar a Soporte](https://www.alkosto.com/contacto)

Si tienes dudas, puedes [contactar con soporte](/docs/contacto).
